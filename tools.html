Tools

IntronDB: a database for eukaryotic intron features

IntronDB catalogs approximately 50,000,000 introns from over 1000 genomes spanning the major eukaryotic clades in the tree of life.

GCevobase: an evolution-based database for GC content in eukaryotic genomes

GCevobase displays GC contents (including GC1, GC2 and GC3) for all annotated coding sequences for 1118 eukaryotic genomes in both intraspecific and interspecific manners.

hppRNA: a Snakemake-based handy parameter-free pipeline for RNA-Seq analysis of numerous samples

hppRNA is a Snakemake-based handy parameter-free pipeline for RNA-Seq analysis of numerous samples with the functionality to work with fusion genes, single nucleotide polymorphisms, messenger RNAs, long noncoding RNAs and circular RNAs.
